emexOS

emexOS is a graphical, from scratch operating system written entirely in C by Voxi0, tsaraki, emexSW. Bi Moz, and Novice it's designed for developers and customization enthusiasts, who want full control over their systems.

Developement: emexOS is still under hard developement and IS NOT daily-drivable yet.

> MAIN FEATURES
Made from scratch
Entirely written from scratch (except the bootloader...)
Limine Bootloader
Limine is a advanced bootloader, excellent for x86-64 architectures
architecture
currently emexOS only supports x64 intel chips but in future maybe more!
drivers
currently there arent much drivers but the most useful one is the ATA driver, i hope i can implement AHCI soon!
console
a small kernel-mode console with 25+ commands
fonts
shoose between multiple pixel-art fonts for the console

Run emexOS on real Hardware:

- Download the iso file

- format a USB with FAT32, use MBR

- flash the iso to the USB (use rufus, balena etcher or `dd`

- boot the OS from the USB

requirements: a x86-64 compatible CPU (Intel), at least 512MB RAM (the more the better), and about 1 GB disk storage is recommended
